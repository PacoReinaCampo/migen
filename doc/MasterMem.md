*MasterMem in Migen*

1. Download Migen
```
git clone https://github.com/m-labs/migen
```

2. Install Migen
```
cd migen
sudo python3 setup.py install
```

3. Synthesize Mem and Simulate MasterMem
```
mv MasterMem.py migen/examples/sim
cd migen/examples/sim
python3 MasterMem.py
```
